# jota


### Roadmap

- [x] Lexing
- [ ] Ast Building (90%)
- [ ] Semantic Analysis
- [ ] Code Generation

### Gettin Started:
```
git clone https://github.com/superPuero/jota
cd jota
make
```

### Using compiler
```
./jota jota_src/test.jo -ast -tokens
```
* **-tokens** dump tokens into stdout
* **-ast** dump abstract syntax tree into stdout
