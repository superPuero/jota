### Roadmap

Lexing done

### Gettin Started:
```
git clone [https://github.com/YOUR_USERNAME/jota.git](https://github.com/YOUR_USERNAME/jota.git)
cd jota
make
```

### Using compiler
```
./jota jota_src/test.jo -ast -token
```
* **-tokens** dump tokens into stdout
* **-ast** dump abstract syntax tree into stdout
